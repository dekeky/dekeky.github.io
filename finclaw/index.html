三步开始

面向最终用户：从 Release 下载，无需安装开发环境。

1

下载并解压

在 Releases 选择 Windows / macOS / Linux 对应包，解压得到 finclaw 可执行文件。

2

运行程序

双击或在终端执行。首次启动会自动创建 ~/.finclaw 数据目录，默认监听 8082 端口。

3

打开浏览器

访问 http://127.0.0.1:8082，注册账号 → 配置模型 → 从市场安装格雷厄姆、巴菲特等大师模板 → 开始对话。
